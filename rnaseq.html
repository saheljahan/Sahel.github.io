DESeq2 Time-course Design

One useful approach for time-course RNA-seq is using an interaction design:

design = ~ sex + condition + time + condition:time
dds <- DESeq(dds)
results(dds, name="conditionTreatment.timeD4")

This helps test whether treatment effects change across time instead of doing overly simple comparisons.
